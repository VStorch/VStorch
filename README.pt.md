### Projetos em destaque

#### **Torch — Sistema de agendamento para Pet Shops**

**Visão Geral**

Sistema full-stack de agendamento para pet shops, projetado para modelar fluxos de negócio do mundo real

O sistema cobre todo o fluxo de negócio, incluindo gerenciamento de usuários, cadastro de pets, agendamento de serviços e controle de horários disponíveis garantindo consistência de dados, aplicação de regras de negócio e uma separação clara de responsabilidades entre camadas.

Inclui funcionalidades como recuperação de senha por e-mail, integração com deep linking e containerização com Docker.

**Tecnologias**

- Java 21, Spring Boot, JPA/Hibernate
- MySQL
- Docker / Docker Compose
- Flutter (cliente mobile)
- HTML/CSS/JavaScript (integração web)

**Principais Contribuições**

- Modelagem de domínio e definição de regras de negócio do sistema de agendamento
- Implementação de camadas de DTO e mapeamento para desacoplamento entre contratos da API e modelos de domínio
- Modelagem relacional com JPA/Hibernate, garantindo consistência entre entidades e relacionamentos
- Implementação da lógica de agendamento, garantindo consistência entre horários disponíveis e agendamentos realizados
- Containerização do backend e banco de dados com Docker Compose para ambientes reproduzíveis
- Produção de documentação técnica (arquitetura, modelos de domínio e fluxo da API)

**Destaques de Arquitetura**
- Arquitetura em camadas (Controller, Service, Repository)
- Uso de DTO para desacoplamento entre API e domínio
- Organização de pacotes voltada à escalabilidade e manutenção

Repositórios:
- Backend: [TorchAppBackend](https://github.com/VStorch/TorchAppBackend)
- App Mobile: [TorchApp](https://github.com/VStorch/TorchApp)
- Página de Redirecionamento: [PaginaRedirecionamento](https://github.com/VStorch/PaginaRedirecionamento)

#### **Pre-Historic Run — Jogo Endless Runner 2D em Unity**  

Jogo 2D do tipo endless runner desenvolvido em Unity, onde o jogador controla um tigre-dentes-de-sabre em cenários pré-históricos com variações sazonais entre verão e inverno.

O projeto foca na implementação de sistemas de gameplay baseados em estados (movimento, obstáculos e pontuação), com ênfase em consistência e fluidez do loop de jogo.

**Tecnologias:** Unity, C#
